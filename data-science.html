Bilingual CAT Tool (PDF to Editable Word)

Provides a free and open-source alternative to commercial Computer-Assisted Translation (CAT) tools. It extracts text from PDFs, segmenting it for easy bilingual translation and presenting it in an editable table format.

Sentimental Analysis using LLaMA 3.1

Leverages in-context learning and Llama 3.1 8B to perform sentiment analysis on movie reviews from the IMDB dataset. It identifies the sentiment expressed within text using advanced NLP techniques.

PDF Summarizer with OpenAI

An AI-powered tool that summarizes PDF documents using OpenAI's GPT models. Upload a PDF, extract the text, and generate a concise summary. Requires an OpenAI API key.

Smart Guide

A personalized Saudi Arabia tourism assistant using OpenAI's GPT-4. Provides detailed recommendations based on user preferences for location, budget, trip type, family composition, and duration, offering tailored travel advice.
