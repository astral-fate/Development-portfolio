Deep Learning Application for Arabic Sign Language (ArSL)

This project aims to utilize the Arabic Sign Language (ArSL) dataset containing 52K essential Arabic alphabets to train a deep learning model, in order to create a computer vision application that assists disabled people to foster inclusive communication.

Classification and Computer vision application of Remote physical therapy exercises

This project aims to utilize the dataset of 10 different PT exercises collected from Infinity AI to train a deep learning model, in order to create a computer vision application that assists the patient while they perform their rehabilitation exercises.
